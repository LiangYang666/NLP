# sohu2021-baseline
2021搜狐校园文本匹配算法大赛baseline

## 简介

分享了一个搜狐文本匹配的baseline，主要是通过条件LayerNorm来增加模型的多样性，以实现同一模型处理不同类型的数据、形成不同输出的目的。

线下验证集F1约0.74，线上测试集F1约0.73。预训练模型是[RoFormer](https://github.com/ZhuiyiTechnology/roformer)，也欢迎对比其他预训练模型的效果。
